## I/O builtins

---

```print(v: any) -> none```<br>
Prints ```v``` to ```stdout``` with formatting, without a newline

---

```println(v: any) -> none```<br>
Prints ```v``` to ```stdout``` with formatting, with a newline

---

```readln() -> string```<br>
Reads ```stdin``` until newline, returns the input
